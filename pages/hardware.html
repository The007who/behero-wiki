Hardware Page

Inner workings of the console:

The console is comprised of two boards, the mainboard and the input board:

Mainboard(side A):

  • AVS641604L-6TE: This is quite possibly RAM.
  • GPR27P512A-006A 3D74270081 (?)102812: A 512M-BIT NAND interface with OTP. This is very likely where the game is stored. The data sheet can be found here.
Mainboard(side B):
  • 324ST e4 N009013: This chip is a LM324, a quad opamp possibly for audio.
  • FT24C08A: It's a 1KB EEPROM made by Fremont. 2.5V to 5.5V. I2C memory interface.
    1KB for the whole game is very unlikely, maybe data only for booting or saves. The data sheet can be found here.
