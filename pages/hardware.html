Hardware Page

Inner workings of the console:

The console is comprised of two boards, the mainboard and the input board:

Mainboard(side A):

  • AVS641604L-6TE: There is a struggle to find information about this specific chip.
  • GPR27P512A-006A 3D74270081 (?)102812: This chip seems to be a TSOP device manufactured by GT. The data sheet can be found here.
Mainboard(side B):
  • 324ST e4 N009013: This seems to be a EEPROM manufactured by STMicroelectronics. The chip is similar to the M95P32-I, which is a low-power 32 Mbit Serial SPI Page EEPROM.
    Next to it there are two test pads, so it likely uses the I2C interface. This is likely where the game data is saved.
  • FT24C08A: It's a 1KB EEPROM with a SPI interface made by Fremont. 2.5V to 5.5V. I2C memory interface.
    1KB for the whole game is very unlikely, maybe data only for booting or saves.
